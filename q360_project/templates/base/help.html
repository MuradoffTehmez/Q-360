İstifadəçi Təlimatı

Sistemə Giriş
  1. İstifadəçi adınızı və şifrənizi daxil edin
  2. İlk girişdə şifrənizi dəyişməyiniz tövsiyə olunur
  3. Dashboard səhifəsinə yönləndiriləcəksiniz
Qiymətləndirmənin Tamamlanması
  1. "Mənim Tapşırıqlarım" bölməsinə keçin
  2. Tamamlanmamış qiymətləndirməni seçin
  3. Bütün sualları cavablandırın
  4. "Göndər" düyməsinə basın
Hesabatların Baxılması
  1. "Hesabatlar" menyusuna keçin
  2. "Mənim Hesabatlarım" bölməsini açın
  3. İstədiyiniz kampaniyanı seçin
  4. Detallı hesabat və qrafiklərə baxın
İnkişaf Məqsədlərinin Müəyyənləşdirilməsi
  1. "Mənim Məqsədlərim" səhifəsinə keçin
  2. "Yeni Məqsəd" düyməsini klikləyin
  3. Məqsədin detallarını daxil edin
  4. İrəliləyişi mütəmadi olaraq qeyd edin
Məsləhət: Hər bir bölmədə "?" ikonu ilə əlavə kömək məlumatlarını tapa bilərsiniz.

Tez-tez Soruşulan Suallar

Qiymətləndirmə

Addım 1: Sol menyudan "Mənim Tapşırıqlarım" bölməsinə keçin.
Addım 2: Aktiv qiymətləndirməni seçin və "Başla" düyməsini klikləyin.
Addım 3: Bütün sualları diqqətlə oxuyub cavablandırın.
Addım 4: "Yadda saxla" ilə qaralama olaraq saxlaya və ya "Göndər" ilə tamamlaya bilərsiniz.

Xeyr, qiymətləndirməni göndərdikdən sonra onu dəyişə bilməzsiniz. Ona görə də göndərmədən əvvəl bütün cavablarınızı yoxlayın. Qaralama rejimində isə istənilən sayda dəyişiklik edə bilərsiniz.

Xeyr, qiymətləndirmələr tam məxfidir. Yalnız anonim olaraq ümumi nəticələrdə istifadə olunur. İndividual cavablarınız heç kim tərəfindən görünməz.
Hesabatlar

Yuxarı menyudan "Hesabatlar" bölməsinə keçin və "Mənim Hesabatlarım" seçin. Burada bütün kampaniyalarınızın nəticələrini və performans trendlərini görə bilərsiniz.

Bəli, hesabat səhifəsinin yuxarı sağ küncündə "İxrac" düyməsi var. Buradan PDF və ya Excel formatında hesabatınızı yükləyə bilərsiniz.
Texniki

Profil menyusundan "Təhlükəsizlik" bölməsinə keçin və "Şifrə Dəyişikliyi" formasını doldurun. Cari şifrənizi və yeni şifrənizi (2 dəfə) daxil edib "Yadda saxla" düyməsini klikləyin.

İnsan Resursları departamenti ilə və ya texniki dəstək komandası ilə əlaqə saxlayın. Onlar sizə yeni şifrə verəcəklər. E-poçt: support@q360.gov.az

Q360 sistemi bütün müasir brauzerlərdə işləyir: Google Chrome, Mozilla Firefox, Microsoft Edge, Safari. Ən yaxşı təcrübə üçün brauzerin son versiyasından istifadə edin.
